# Multi Tenant Blog Platform

## Development Purpose
- `runDev.ps1` - This runs both the script to start frontend and backend, then opens the application in GOOGLE CHROME browser. 

## Frontend
- `npm install` - Install all dependencies listed in package.json
- `npm start` - Run the start script defined in package.json
- `npm run dev` - Common command for running development servers

## Backend
- `uvicorn main:app --reload` - Run FastAPI with auto-reload for development
